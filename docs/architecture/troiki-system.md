# Система «Тройки»

Тройки – это стратегическая система работы с важными задачами, на горизонте от недели и более.

Три – это очень удобное для человеческого восприятия число, которое легко запомнить.

В основе системы лежит ограничение по числу 3:
1. У вас есть три категории:
  - Важное
  - Среднее
  - Остальное
2. В каждой категории можеть быть максимум 3 задачи, у каждой из этой задач может быть максимум 3 подзадачи и так далее.

Каждая из категорий влияет на ту что ниже. «Важное» на «Среднее», а «Среднее» на «Остальное». При этом «Важное» никак не влияет напрямую на «Остальное».

Как только вы выполнили задачу из «Важного», то появляется возможность добавить 1 одну задачу в категорию «Среднее», если там менее 3х задач. Тоже относится и к отношению между «Средним» и «Остальным».

## Рекомендации

- Работаем над задачами по правилам GTD:
